The main type of document on the web is called HTML- Hypertext markup language. HTML consist of 4 different things:

Text content (what you see)
Markup (what it looks like)
Reference to another document (e.g. images+videos)
Links to other pages

The markup is what we use to change the normal text into what we would like it look like.
